Automatically apply a coupon code when a customer's cart subtotal exceeds a defined amount.

== Description ==

This plugin adds a simple WooCommerce setting to automatically apply a coupon if the cart total exceeds a certain amount. Great for promotions and offers.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/auto-apply-coupon-amount/` directory, or install it through the WordPress plugins screen.
2. Activate the plugin.
3. Go to WooCommerce > Settings > Auto Apply Coupon to configure.

== Frequently Asked Questions ==

= Does it support multiple coupons? =
Currently, it supports one auto-applied coupon.
